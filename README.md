# Java Fundamentals Lab

A personal repository dedicated to practicing and learning Java SE through exercises, algorithms, and small projects.

## Objective

To solidify Java fundamentals through practical exercises, unit testing, and best development practices, as preparation for backend development with Spring Boot and integration technologies.

## Technologies

- Java 17
- Maven
- JUnit 5

## Contents

- Object-Oriented Programming
- Collections
- Exception Handling
- Input/Output (I/O)
- Java Streams and Lambda Expressions
- Multithreading and Concurrency
- JDBC
- Algorithms

  ## Project Structure
![Image text](https://github.com/TanqDev88/javaLab/blob/main/ProjectStructure.png)

The exercises are organized by topic, and each one includes its corresponding unit tests using JUnit.

## How to Run the Tests

```bash
mvn test
```

## Requirements

- JDK 17 or higher
- Apache Maven 3.9+
- IntelliJ IDEA (recommended)
